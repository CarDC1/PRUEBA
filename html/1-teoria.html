1. Teoría

Libro de texto

El contenido teórico con el que trabajaremos en las diferentes actividades de esta Situación de Aprendizaje se corresponde con el tema 4 del libro de texto (pág. 144).

Además, aquí podréis encontrar otras fuentes de información relacionadas con el tema en distintos formatos que os ayudarán a todos y a todas a comprender mejor los conceptos más teóricos. 

Para la realización de las tareas será necesario que dediquéis un tiempo en vuestra casa a leer y a ver toda esta información, y podamos aprovechar mejor el tiempo de las clases realizando las tareas con el ordenador.

Diccionario de términos

Sitio web: formado por un conjunto de páginas web enlazadas entre sí, a las cuales se accede a través de una dirección común.

Gestor de contenidos: es una herramienta de publicación que permite generar contenidos web y editarlos y actualizarlos online sin necesidad de contar con conocimientos técnicos de diseño o programación.

Lenguaje HTML: es el lenguaje con el que se crean las páginas web.

Editores de páginas web: son aplicaciones que permiten crear páginas web.

FTP (Programas de Transferencia de Ficheros): son los encargados de comunicar el ordenador local con el servidor que brinda alojamiento.

Maquetación: se encarga de organizar espacios y contenidos escritos, visuales y audiovisuales.

Interfaz: es el conjunto de comandos y métodos que facilitan la interacción entre el usuario y el sistema.
