1.4 El lenguaje HTML

La teoría está en la página 150. LEER

Resumen

HTML es el lenguaje básico con el que se construyen las páginas web. Sus siglas significan HyperText Markup Language (Lenguaje de Marcado de Hipertexto), y sirve para estructurar el contenido que vemos en Internet: textos, imágenes, enlaces, tablas, formularios, entre otros.

A diferencia de los lenguajes de programación, HTML no “programa” acciones complejas, sino que organiza y describe la información mediante etiquetas. Estas etiquetas indican al navegador (como Chrome o Firefox) cómo debe mostrarse cada elemento en la pantalla.

Un documento HTML se compone de una serie de etiquetas escritas entre corchetes angulares, por ejemplo:

. Normalmente, las etiquetas funcionan en pares: una de apertura y otra de cierre. Por ejemplo:

Este es un párrafo

La etiqueta

lo cierra. Todo lo que esté dentro se considera contenido del párrafo.
